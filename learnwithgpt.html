Learn With GPT

Web App Development

Overview

With a team of 4 software engineers, we developed a web application that extends ChatGPT's capabilities, facilitating unlimited memory and enabling the loading of external data including but not limited URLs, PDFs, Word documents, and PNG files. Our solution employs LlamaIndex for enhancing Large Language Model performance. Key features include an uploader for data import, a chatbot that uses chat history to retain memory, and a button for generating one-click formatted summaries.

Timeline: 4 weeks

Design: Figma

Frontend: Typescript + React + Node + Ant Element

Backend: Python + Django

Large Language Model: OpenAI + LlamaIndex + LangChain

Context: Brown CS1320 Software Project

Context

Faced with information overload, cross-referencing challenges, and content accessibility issues in daily and professional life, a diverse user market comprising software engineers, financial analysts, and college students requires an efficient, automated solution. We've developed an AI-enhanced web application that leverages the power of Large Language Models, specifically OpenAI's ChatGPT to automate data processing from multiple sources and generate summaries for easier comprehension, all while retaining unlimited memory. our solution streamlines workflows, accelerates learning, and enhances the information accessibility across different domains.
